🍱 **赛博社交场：山东饭局情商模拟器**

这是一个基于 AI 大模型的网页端“情商训练工具”，旨在通过高度还原的山东饭局场景，助你肉身练级，成就酒桌战神！

🌟 **项目简介 (Project Intro)**

本项目是一款兼具**娱乐性**与**实战性**的社交模拟工具。我们以“山东饭局”这一极具仪式感的社交符号为切入点，利用 AI 模拟棘手的饭局沟通局面。

*   **解决痛点：** 缓解社交焦虑，纠正低情商行为，让用户在低风险的虚拟环境中反复试错。
    
*   **核心立意：** 借 AI 之力，赋能真实社交。
    

🛠️ **核心功能 (Core Functions)**

*   🎭 **动态场景生成：** 预设家庭聚会、商务宴请、同学会等剧本，AI 实时生成性格各异的 NPC。
    
*   🎙️ **实时语音交互：** 支持 ASR 语音输入，模拟真实饭局中的即时反应压力。
    
*   🥞 **煎饼与大蒜系统：** 
    
    *   **Pancake (煎饼)：** 表现得体，积分增加。
        
    *   **Garlic (大蒜)：** 触发禁忌，获得嘲讽。
        
*   🆘 **召唤山东人救场：** 当局势陷入僵局，点击按钮获取“酒桌天花板”给出的高情商标准答案。
    
*   📊 **赛博复盘报告：** 对局结束生成多维社交雷达图，授予“酒桌老狐狸”或“饭桌木头人”等称号。
    

✨ **项目亮点 (Highlights)**

*   **深度融合：** 拒绝“为了 AI 而 AI”。AI 深度参与判分、逻辑推进与 NPC 内心 OS，构建完整的反馈闭环。
    
*   **梗文化与硬核逻辑结合：** 将网传的“山东情商梗”转化为可量化的判分模型，极具病毒式传播潜力。
    
*   **多 Agent 协作：** NPC 之间有立场、有矛盾、有互动，模拟真实的复杂人际场。
    
*   **全场景扩展潜力：** 底层框架支持迁移至面试、答辩、职场汇报等正式沟通场景。
    

⚙️ **技术架构 (Technical Implementation)**

*   **部署平台：** ModelScope (魔搭)
    
*   **前端：** Web 页面负责场景配置、对话展示、积分与动画反馈
    
*   **后端/AI 逻辑：**
    
    *   **任务解耦：** 采用 **“异步串行链路”**，将“裁判判分”与“编剧接话”拆分为两个原子任务，确保在有限算力下实现秒级反馈。
        
    *   **结构化 Prompt：** 通过强制 JSON 输出约束，确保逻辑判定与 UI 动效的精准绑定。
        
    *   **记忆管理：** 维护多轮对话的上下文，保持 NPC 人设的一致性。
